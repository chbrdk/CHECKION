# Report export templates

CHECKION executive report export assets.

| File | Purpose |
|------|---------|
| `MSQDX_PPT-Master_27-05-26.pptx` | Corporate PowerPoint master (16:9, 71 layouts) |
| `PNG/MSQDX_Logotype_White.png` | Logo on dark backgrounds |
| `PNG/MSQDX_Logotype_Black.png` | Logo on light backgrounds |
| `SVG/` | Vector logotypes (optional) |
| `brand-tokens.json` | Colors and fonts for PptxGenJS masters |

See:

- `knowledge/checkion-report-pptx-export-design.md`
- `knowledge/msqdx-ppt-master-layout-mapping.md`

Override paths via env:

- `CHECKION_REPORT_PPTX_MASTER_PATH`
- `CHECKION_REPORT_PPTX_LOGO_WHITE_PATH`
- `CHECKION_REPORT_PPTX_LOGO_BLACK_PATH`
- `CHECKION_REPORT_PPTX_BRAND_TOKENS_PATH`
